# Quick Start Guide

## üöÄ Get Started in 5 Minutes

### 1. Install Extension (2 minutes)

1. Open Chrome ‚Üí `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `Extension_elementer` folder
5. **Important:** Add icon files to `icons/` folder (see icons/README.md)

### 2. First Use (1 minute)

1. Go to any website (e.g., `example.com`)
2. Click extension icon in toolbar
3. Click "Start Converting"
4. You should see overlay toolbar appear

### 3. Select Elements (1 minute)

1. Hover over page elements (they'll highlight in blue)
2. Click elements to select (they'll show red outline)
3. Watch selection counter update

### 4. Analyze & Export (1 minute)

1. Click "Analyze" button in overlay
2. Click "Export" ‚Üí Choose format
3. Download your Elementor JSON!

## üéØ Common Workflows

### Workflow 1: Quick Section Copy

1. Activate extension
2. Click on section/container
3. Click "Analyze"
4. Export as JSON
5. Import to Elementor

### Workflow 2: Full Page Conversion

1. Activate extension
2. Select multiple sections
3. Analyze all
4. Export as JSON
5. Review in Inspector panel
6. Import to Elementor

### Workflow 3: Style Reference

1. Select element
2. Open Inspector panel
3. Go to "Styles" tab
4. Copy specific styles
5. Apply manually in Elementor

## üÜò Quick Troubleshooting

**Extension not working?**
- Check Chrome version (needs 88+)
- Reload extension
- Check browser console (F12)

**Selection not working?**
- Ensure page is fully loaded
- Try different website
- Check for iframe restrictions

**Export failing?**
- Analyze elements first
- Check browser console
- Verify elements are selected
