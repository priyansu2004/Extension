## üöÄ How to Use

### Step 1: Activate Extension
1. Click the extension icon in Chrome toolbar
2. Click **"Start Converting"** button
3. You should see:
   - Overlay toolbar at the top
   - Elements highlight in blue when you hover
   - Tooltip shows element info

### Step 2: Select Elements
- **Hover** over any element ‚Üí It highlights in blue
- **Click** an element ‚Üí It gets selected (red outline)
- **Click again** ‚Üí Deselects it

### Step 3: Export to JSON

#### Option A: Export Hovered Element (Quick)
1. Hover over the element you want
2. Click **"Export Hovered"** button in toolbar
3. JSON file downloads instantly!

#### Option B: Export Selected Elements
1. Click elements to select them (they show red outline)
2. Click **"Export JSON"** button in toolbar
3. All selected elements exported as JSON

#### Option C: Analyze First (Detailed)
1. Select elements
2. Click **"Analyze"** button
3. View detailed analysis in sidebar
4. Then export
