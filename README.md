# Extension Icons

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Creating Icons

You can create simple icons using:

1. **Online Tools:**
   - Canva
   - Figma
   - GIMP
   - Photoshop

2. **Quick Placeholder:**

3. **Icon Libraries:**
   - Flaticon
   - Icons8
   - Font Awesome

## Recommended Design

- Use Elementor brand colors (purple gradient)
- Keep design simple and recognizable
- Ensure icons are clear at small sizes
- Use transparent background (PNG)

## Testing

After adding icons:
1. Reload extension in Chrome
2. Verify icons appear correctly
3. Check all three sizes display properly
